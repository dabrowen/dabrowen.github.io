Projects

What I created in my free time

  • Macro

    Inspired by my journey of bodybuilding, diet has been seen as an integral aspect of losing weight and building muscle. As nutritional information is rarely taught in grade school, many are unaware of the number of calories or macronutrients in everyday food. Macro was created as an easy tool for others to find out the nutritional information or food with just a click of a finger. The application utilizes the Google Cloud Vision API and queries through the USDA food composition database to obtain nutritional info and display on the application. The application also utilizes Firebase to allow users to store their data and recent logs. The front end of the application was created through React Native allows for dual compatibility with Android and IOS devices.

    Check it out!

  • Shared Paper

    Due to a lack of a collaborative platform that meets the demands of workplace productivity, artistic creativity, and educational connectivity, Shared Paper was created as a synchronized, multi-purpose canvas for people to work on. To simulate a synchronous system like Google Drive, Shared Paper utilizes MongoDB, Socket.io, and Node.js to communicate data through multiple devices and a server to allow users to join documents and experience real time synchronization. The front end of the application was created exclusively through React Native but is currently only compatible for IOS devices.

    Check it out!

  • Flow - 2019 EqualsEquals Capital One Best Financial Hack

    One of my first projects using React Native, an open-source mobile application framework created by Facebook to develop applications for Android and IOS. This mobile application is simple but powerful as it allows easy tracking for income and expenses with a few button clicks, immensely useful for anyone.

    Check it out!
